# Sistema de gerenciamento de ativos de ti e vulnerabilidade 

# descrição

# Este projeto  foi desenvolvido  em pyhton com o objetivo de gerenciar ativos de tecnologia e suas vulnerabilidades associadas.
# O sistema permite cadastrar, listar, atualizar e remover ativos, e registrar vulnerabilidades com diferentes níveis de severidade e status.
# Os dados são armazenados em um arquivo json.
